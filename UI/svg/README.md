# SVG

- svg는 일반 DOM과 다르게, css가 우선순위가 더 높다.
- svg 태그안에 스타일태그와 스크립트태그를 이용해서 스타일 스크립트 적용 가능

- fill : svg 태그 내부에 채울 색
- stroke: 테두리 지정
- stroke-width: 테두리 너비
